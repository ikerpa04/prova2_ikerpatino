# Introducció
## Qué es Markdown
Markdown es un lenguaje de marcado ligero creado por John Gruber y Aaron Swartz que trata de conseguir la máxima legibilidad y facilidad de publicación tanto en su forma de entrada como de salida, inspirándose en muchas convenciones existentes para marcar mensajes de correo electrónico usando texto plano


# Sintaxi Básica
## Titols
Para cambiar el tamaño del título tienes que añadir en el inicio de cada línea una almohadilla (#). Cuantas mas almohadillas, mas pequeño será el tamaño del texto.
Ejemplos:
> # Titulo1
> ## Titulo2
> ### Titulo3


## Paragrafs
Para añadir paragrafos hay que hacer doble intro y añadiendo dos almohadillas para su titulo.

## Estils
### Negreta y Cursiva
Para añadir efecto negrita en el texto, hay que añadir dos asteriscos ( ** ) delante y detras del texto (**hola**).

Para a cursiva, solo hay que añadir una (*) (*hola*).

Para añadir las dos a la vez, habra que poner los 3 asteriscos (*), (***hola***)

## Cites en bloc
Para crear una cita de bloque, hay que añadir al principio de cada línea una "flecha" (>)
> hola

Para crear varías con dar intro estaría correcto
> Hola
>
> Hola

Y para hacer un subbloque añadir otra flecha
> Hola
>
> > Hola

## Enllaços
Para añadir enlaces, en una palabra, en este caso YouTube, hay que ponerlo así
> [YouTube] (https://www.youtube.com)
> [YouTube](https://www.youtube.com)

Para añadir enlaces enteros, simplemtente poniendolo entre () seria suficiente.
> (https://www.youtube.com)

Y para los correos electronicos seria mas de lo mismo
> estecorreonovale@novale.com

## Imatges
Para añadir imagenes hay que añadir un signo de exclamación (!),seguidamente de un texto alternativo entre corchetes [], continuado del enlace de la imagen.

# Sintaxi Extesa
## Tablas
Para crear tablas, hay que añadir tres o mas guiones (---) para crear el encabezado de cada columna y para separar cada columna, hay que usar barras verticales (|).
> |hola |holaaa |
> |-----|-------|
> |hola |holaaaa|
> |hola |holaaaa|
